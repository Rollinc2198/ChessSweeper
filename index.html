ChessSweeper Rules

  1. Normal Chess rules apply

  2. Normal Minesweeper rules apply

    • The first square clicked is not guaranteed to be empty

    • Each player has the same Minesweeper board

  3. If you reveal a bomb, you lose your turn

  4. If you place a piece on a bomb, you lose that piece

  5. If a piece is removed from the Chess board, the Minesweeper board is reset
